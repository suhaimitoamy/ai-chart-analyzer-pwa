# XAUUSD ICT Chart Analyzer PWA

Versi Web App / PWA dari aplikasi XAUUSD ICT Chart Analyzer.

## Fungsi

- Real-time XAU/USD dari Twelve Data WebSocket
- Gold Price live
- WIB Time live
- Session otomatis dengan timezone asli dan DST
- CandleBuilder multi-timeframe
- Analisis ICT saat tombol diklik
- Rule-based SMC engine lokal
- Terminal Market Event Feed
- History Trade berdasarkan TP/STOP
- PWA bisa dibuka di Android, iPhone, Windows, Mac, dan browser

## Cara menjalankan lokal

```bash
npm install
npm run dev
```

## Cara build

```bash
npm run build
```

## Deploy

Bisa deploy ke Vercel, Netlify, atau GitHub Pages.

## Catatan

Aplikasi ini bukan auto-trade. Aplikasi hanya membantu mapping market dan membaca event market.
